Se desarrolla utilizando los métodos nativos de JavaScript para obtener el número de la letra con valor.charCodeAt(valor) + desfase, cambiar el valor y volver a obtener el valor usando String.fromCharCode() Para obtener el resultado de valor ASCII desfasado a letra, se usa el método String.fromCharCode(valor) que toma como argumento un número que representa el código ASCII y devuelve la letra correspondiente. Luego, se imprime el resultado alterando el contenido de la etiqueta HTML que tiene asignado un identificador único utilizando el método document.querySelector(id).innerHTML = resultado en pantalla
